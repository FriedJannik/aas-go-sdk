# AAS Go SDK V3.1

Go SDK for the Asset Administration Shell (AAS) V3.1 metamodel, auto-generated using [aas-core-codegen](https://github.com/aas-core-works/aas-core-codegen).

## Features

- **Full V3.1 support** - Generated from the official [aas-core-meta](https://github.com/aas-core-works/aas-core-meta) V3.1 metamodel
- **Proper interfaces** - `IClass`, `ISubmodelElement`, `IReferable`, `IIdentifiable`, etc.
- **Type-safe** - Go interfaces model the AAS inheritance hierarchy
- **Verification** - Built-in constraint checking
- **JSON/XML serialization** - De/serialize AAS models
- **Enhancing** - Attach custom data to model instances

## Usage

```go
package main

import (
    "fmt"
    aastypes "github.com/FriedJannik/aas-go-sdk/types"
    aasjsonization "github.com/FriedJannik/aas-go-sdk/jsonization"
    aasverification "github.com/FriedJannik/aas-go-sdk/verification"
)

func ptr[T any](v T) *T { return &v }

func main() {
    // Create a Property
    property := aastypes.NewProperty(aastypes.DataTypeDefXSDString)
    property.SetIDShort(ptr("myProperty"))
    property.SetValue(ptr("Hello, AAS!"))

    // Create a Submodel
    submodel := aastypes.NewSubmodel("https://example.com/submodel/1")
    submodel.SetSubmodelElements([]aastypes.ISubmodelElement{property})

    // Create an Environment
    env := aastypes.NewEnvironment()
    env.SetSubmodels([]aastypes.ISubmodel{submodel})

    // Verify constraints
    aasverification.Verify(env, func(err *aasverification.VerificationError) bool {
        fmt.Printf("Error: %s\n", err.Error())
        return false // continue checking
    })

    // Serialize to JSON
    jsonable, _ := aasjsonization.ToJsonable(env)
    fmt.Printf("%+v\n", jsonable)
}
```

## Packages

| Package | Description |
|---------|-------------|
| `types` | All AAS interfaces and structs (`IClass`, `ISubmodel`, `IProperty`, etc.) |
| `constants` | Constant values and enum sets |
| `jsonization` | JSON serialization/deserialization |
| `xmlization` | XML serialization/deserialization |
| `verification` | Constraint verification |
| `enhancing` | Attach custom data to instances |
| `stringification` | Convert enums to/from strings |
| `reporting` | Error path reporting |
